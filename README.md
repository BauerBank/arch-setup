## How to use
To start enter:

<code>chmod u+x manjaro-setup.sh</code>

and than execute it:

<code>sudo ./arch-setup.sh user</code>


for user you have the fill in your corrent user to do the stuff you dont whant to do es a root user. For me it is the AUR part
